Nome de banco de dados: dbBanco;
Nome de variavel no sql: idVariavel, nomeVariavel;
Nome de foreign key: fk_idVariavel;

Nome de function: function();

Não utilizar '_' no começo de nomes de funções;
Não usar acentuação;
